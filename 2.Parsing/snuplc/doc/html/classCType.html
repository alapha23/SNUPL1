◆ Match()

virtual bool CType::Match ( const CType *  t) const
pure virtual

match two types

Match() and Compare() differ as follows: Match() returns true for compatible types whereas Compare() only returns true for identical types. Except for array and pointer types, Match() and Compare() return the same result.

Parameters
ttype to compare this type to
Return values
