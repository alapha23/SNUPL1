◆ Match()

bool CArrayType::Match ( const CType *  t) const
virtual

compare two types

CArrayType::Match() matches this type with type . The matching is performed recursively on each dimension down to the base type. In each dimension, the number of elements must match or 's dimension be CArrayType::OPEN (but not vice-versa, i.e., when type matching array function arguments with function parameters, the argument must call Match() with the formal parameter as its argument.

ttype to compare this type to
Return values
